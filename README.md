# TREASURE-ISLAND
find the treasure!!

This is a simple text-based adventure game written in Python.

The player makes choices to find the hidden treasure.  
Different choices lead to different endings.

## Game Description

You start your journey on Treasure Island and must choose the correct path to find the treasure.

Make the right decisions to win the game!


## Example

Welcome to Treasure Island

You're at a crossroad.
Type "left" or "right"

left

You reached a lake.
Type "wait" or "swim"

wait

Choose a door: red, blue, or yellow

yellow

You found the treasure! You Win!
